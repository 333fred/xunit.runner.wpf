# xunit.runner.wpf
XUnit Gui written in WPF

A simple replacement for the old winforms xunit.gui with support for xunit 2.0.

Find it on NuGet at [xunit.runner.wpf](https://www.nuget.org/packages/xunit.runner.wpf).
